Contact form

Descrizione

Ho sviluppato questo modulo di contatto per migliorare le mie competenze nell’utilizzo di CSS Grid. Fondamentale è stato l'uso di JavaScript per gestire interazioni dinamiche, come il cambiamento del colore di sfondo dei campi di input quando vengono selezionati. Per garantire la massima adattabilità, ho implementato un layout a una colonna per dispositivi mobili e un layout a due colonne per tablet e laptop, utilizzando le media query. Il risultato è un modulo di contatto reattivo che mostra un messaggio popup quando viene inviato.
